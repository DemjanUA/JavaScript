2.2 Длина cтроки, свойство length

Одно из самых частых действий со строкой – это получение ее длины:

Длина строки – это количество 16-битных значений, содержащихся в ней. Нумерация символов в строках (и элементов в массивах) в языке JavaScript начинается с нуля: первое 16-битное значение находится в позиции 0, второе – в позиции 1 и т.д. 

'use strict';
var str = 'newStrig';
console.log(str.length); //8

2.5 Доступ к символу

Чтобы получить символ, используйте вызов charAt(позиция). Первый символ имеет позицию 0: 

'use strict';
var str = "jQuery";
alert( str.charAt(0) ); // "j"

В JavaScript нет отдельного типа «символ», так что charAt возвращает строку, состоящую из выбранного символа.

Также для доступа к символу можно также использовать квадратные скобки: 

'use strict';
var str = "Hello worl!";
alert( str[0] ); // "H"

Вообще же метод charAt существует по историческим причинам, ведь квадратные скобки – проще и короче.

Для поиска подстроки есть метод indexOf(substr, from), где substr - подстрока, from - начальная позиция.

2.7 Взятие подстроки

В JavaScript существуют 3 метода для взятия подстроки, с небольшими отличиями между ними.

2.7.1 Метод substr

substring(from [, to]) метод возвращает подстроку с позиции from до, но не включая to.

2.7.2 Метод substring

substr(from [, value]) метод возвращает подстроку с позиции from в количестве символов value

2.7.3 Метод slice

slice(from [, to]) метод возвращает часть строки от позиции start до, но не включая, позиции end. Смысл параметров – такой же как в substring.

Различие между substring и slice – в том, как они работают с отрицательными и выходящими за границу строки аргументами

2.8 Кодировка Unicode

String.fromCharCode(code) - Возвращает символ по коду code

'use strict';
alert( String.fromCharCode(1072) ); // 'а'

str.charCodeAt(pos) - Возвращает код символа на позиции pos. Отсчет позиции начинается с нуля.

'use strict';
alert( "абрикос".charCodeAt(0) ); // 1072, код 'а'
